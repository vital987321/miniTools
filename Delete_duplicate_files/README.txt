This application search for duplicate file and removes them.
Task
Input data from user:
Address to Folder 1
Address to Folder 2

Application checks if there are the same files (based on names) in both folders. If there are application gives the list of files and propose to delete them from folder 1.
Additionally: checks the date of file modification and propose to remove the older version from both folders.
Additionally: search through nested folders. This task to be described better
Additionally: Only one folder for input and check for duplicates in nested folders.
Additionally: instead of deleting files app moves it to temp storage with logging old address for every file. Than user to check results and to remove this folder manually.
